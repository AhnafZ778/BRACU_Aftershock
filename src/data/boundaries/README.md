# Boundary Data

Admin boundary GeoJSON files for the pilot geography.

## Source
- [HDX Bangladesh Admin Boundaries](https://data.humdata.org/dataset/cod-ab-bgd)
- Upazila level (admin3) boundaries for Cox's Bazar district
